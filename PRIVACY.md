# Privacy Policy for Incognito Lite

Incognito Lite does not collect, transmit, sell, or share any personal data.

The extension uses Firefox extension APIs locally on your device to detect page visits and remove matching local history entries while the extension is enabled. While pause mode is enabled, the extension also displays a local visual border on normal web pages so the enabled state is visible.

Data handled locally:

- The current pause/unpause setting is stored in Firefox local extension storage.
- Visited URLs may be read locally by Firefox extension APIs only so they can be removed from local browser history while pause mode is enabled.

Incognito Lite does not send browsing history, URLs, settings, analytics, or any other data to any server.

Incognito Lite is not a full private browsing mode. Websites, search engines, internet service providers, DNS providers, workplace or school networks, and account-level history systems may still record activity outside of Firefox local browser history.
